# RuleYourUsage

For all those who seek to vibe code but still have a thorough understanding of their codebase.

## Description

RuleYourUsage is a tool designed to help developers maintain a comprehensive understanding of their codebase while keeping the development process smooth and enjoyable. Whether you're working on a small project or a large-scale application, RuleYourUsage provides insights into how your code is structured and used.

## Features

- Track code usage patterns
- Maintain codebase awareness
- Simple and intuitive interface
